# Security Policy

## Supported Versions

This project is experimental and does not yet have formal releases. Security fixes should target the default branch unless release branches are introduced later.

## Reporting A Vulnerability

If you find a security issue, please avoid publishing exploit details before the maintainer has had a reasonable chance to respond.

Useful report information:

- Affected file or command.
- Steps to reproduce.
- Impact.
- Suggested fix, if known.

## Safety And Engineering Disclaimer

This repository contains simulation and optimization tools. It does not certify equipment safety, mining recovery, environmental compliance, or manufacturability.

Before building or operating any physical device based on this project:

- Validate with appropriate CFD/DEM and bench tests.
- Use safe mechanical design practices.
- Follow local laws and environmental rules.
- Consider pressure, flow, structural, electrical, and material-handling hazards.

Simulation output should be treated as experimental evidence, not as a guarantee of real-world behavior.
