# Code of Conduct

## Our Pledge

We want this project to be welcoming, respectful, and useful to people interested in simulation, mineral processing, open engineering, and learning.

Participants are expected to:

- Be respectful and constructive.
- Assume good intent when possible.
- Focus criticism on ideas, code, data, and methods rather than people.
- Clearly distinguish experimental results from validated engineering claims.
- Avoid harassment, discrimination, insults, or hostile behavior.

## Scope

This code of conduct applies to project spaces such as issues, pull requests, discussions, documentation, and other collaboration channels connected to this repository.

## Unacceptable Behavior

Examples of unacceptable behavior include:

- Harassment or discriminatory language.
- Personal attacks.
- Publishing private information without permission.
- Deliberately misleading claims about simulation accuracy, safety, or recovery performance.
- Repeated disruption of technical discussion.

## Enforcement

Project maintainers may remove comments, close issues, reject contributions, or restrict participation when behavior is inconsistent with this code of conduct.

If you need to report a concern, open a private communication channel with the maintainer if one is available, or use GitHub moderation/reporting tools.

## Attribution

This is a concise project-specific code of conduct inspired by common open-source community guidelines.
